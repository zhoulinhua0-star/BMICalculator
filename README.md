# BMICalculator
A pragmatic BMI calculator that could accurately calculate your BMI and provide personal health advice to users based on the given height and weight information.

# Project 1: BMI Calc (Console version)
This project is a simple console-based BMI (Body Mass Index) calculator written in Java.
It allows users to calculate their BMI using either the Metric system (cm, kg) or the US system (inches, lbs).
